# Lesson4Lite

Этот проект содержит учебные модули на Python, разработанные в рамках курса.

## Модули

### Модуль 1: myfunctions.py
Содержит реализации различных функций для работы с разделителями, приветствиями, математическими операциями и фильтрацией последовательностей.

### Модуль 2: borndayforewer.py
Программа-викторина о дате рождения А.С. Пушкина. Пользователь должен угадать год и день рождения.

### Модуль 3: use_functions.py
Программа "Личный счет" - симулятор банковского счета с возможностями пополнения, совершения покупок и просмотра истории.

## Установка и запуск

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/todmiv/Lesson4Lite.git
   cd Lesson4Lite
   ```

2. Установите зависимости (если есть):
   ```
   pip install -r requirements.txt
   ```

3. Запустите нужный модуль:
   ```
   python myfunctions.py
   python borndayforewer.py
   python use_functions.py
   ```

## Требования

- Python 3.6+
